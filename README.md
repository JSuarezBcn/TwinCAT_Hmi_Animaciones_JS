# Ejemplo de efectos de animación en javascript para proyectos de TwinCAT HMI

## Descripción

Proyecto de TwinCAT HMI con diferentes funciones javascript y estilos css para realizar animaciones

![image](./capturas/Captura\_00.png)

## Funciones

* Animaciones:
  * rotación de un objeto con control de velocidad
  * rotación de un objeto con control grados de rotación
  * menú deslizante
  * cuenta atrás
  * menú deslizante de contenido tipo 'carrusel'
  * Animaciones de movimiento y sombra movers Xplanar

&#x20; \* Animaciones de movimiento y sombra de movers Xplanar
